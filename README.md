## 为什么你需要这个 Skill？

工作三五年后，很多人会发现一个残酷的事实：**和你同期入职的人，有人已经年薪翻了三倍，而你还在原地焦虑。**

差距不在于谁更勤奋，而在于谁先看清了职场的底层逻辑。

《真本事》这本书的核心洞见是：**个人价值由市场需求决定，而不是由你的努力程度、技能难度或工作年限决定。** 大多数人用错了衡量标准，结果越努力越迷茫。

这个 Skill 做的事，就是把书中这套"看清现实 → 找到杠杆 → 精准行动"的方法论变成你随时可以调用的 AI 顾问。

## 它能帮你做什么？

当你向 Claude 描述一个真实的职场问题，真本事顾问会自动启动**三步诊断**：

**第一步：戳破思维盲区**
书中总结了三种让人原地打转的思维陷阱——你可能一个都没意识到：

- **客体思维**："老板不公平"、"大环境太差"—— 把所有问题推给外界，自己永远是受害者
- **固定型思维**："我天生不擅长这个"—— 用天赋论给自己设限，遇到挑战就退缩
- **打分思维**："他薪资比我高，所以他比我强"—— 用单一数字看复杂世界，结果永远在别人的评分体系里打转

**第二步：匹配书中框架**
根据你的问题类型，从七大核心框架中精准匹配：

| 你的困境 | 适用框架 | 核心思路 |
|---------|---------|---------|
| 要不要跳槽/换行业 | **道天地将法** | 源自《孙子兵法》——选择赛道比埋头苦干重要100倍 |
| 忙到飞起却不出成果 | **10%核心价值** | 工作的价值差异不是1.2倍，而是几十上百倍 |
| 想升职加薪没门路 | **个人价值公式** | 价值 = 了解市场 × 打造产品 × 利用杠杆 |
| 想做副业不知从何开始 | **手艺思维** | 赚钱是一门手艺：行动6分 + 思考3分 + 学习1分 |
| 想实现财务自由 | **财务自由公式** | 被动收入 > 支出，四个变量你要先动哪一个？ |
| 觉得自己没什么核心竞争力 | **核心能力 vs 具体技能** | 编程语言会过时，但"创造"和"销售"永不贬值 |
| 和老板关系不好/没资源 | **杠杆原理** | "嫡系"是结果不是原因——信任是挣来的 |

**第三步：给出行动清单**
不是"加油努力"这种废话，而是具体到你明天就能做的2-4件事。

## 看个例子

**你说：**

> 我在公司做了5年数据分析，考核年年不错，但就是不升职。想做副业，又觉得自己没什么拿得出手的本事。

**真本事顾问会这样帮你：**

1. **诊断**：指出两个隐藏的思维陷阱——"不升职是因为老板不公"是客体思维；"我没什么本事"是固定型思维
2. **框架**：用个人价值公式帮你看清——你5年积累的数据分析经验本身就是市场资产，问题不是"没有本事"，而是你的劳动成果还停在"投入层"（我做了X个报告），没有升级到"成果层"（我帮公司节省了X万成本）
3. **行动**：
   - 下周周报里，把工作描述从"完成了Q1数据报告"改成"通过Q1数据分析，发现并推动了X流程优化，预计年节省Y万元"
   - 列出你比普通人更擅长的3个数据分析技能，这周找2个朋友问他们愿不愿意付费获取这种帮助
   - 主动约你老板聊一次，对齐晋升标准——把"要资源"讲成"能带来增量回报"

## 书中最颠覆认知的几个观点

这个 Skill 背后的书，充满了反直觉但极其实用的洞见：

**"你是动物园的老虎，还是丛林的老虎？"**
很多人在大公司待久了，以为自己的能力值得高薪——其实离开公司平台，在真实市场上可能一文不值。定期在市场上验证自己的价值，是保持竞争力的唯一方式。

**"学阿拉伯语"的思想实验**
如果一个人花10年苦学阿拉伯语，但市场不需要这个技能——他的价值就是零。价值不等于难度，也不等于努力。这是大多数人一辈子都没想明白的事。

**"嫡系是结果，不是原因"**
你以为别人升职是因为"会拍马屁"、"是嫡系"。但真相是：信任和资源是用价值"指"出来的，先展示价值、建立信任，"嫡系"的地位自然到手。

**"成功 ≈ 手艺 + 坚持，没有别的"**
当手艺质量足够高时，机会会主动找上门——因为高质量手艺是稀缺品。你要做的不是到处社交、刷存在感，而是安静地把一件事做到极致。
