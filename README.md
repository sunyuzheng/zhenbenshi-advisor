# 真本事顾问 Skill

基于《真本事：从会工作到会赚钱》（孙煜征著）提炼的 Claude AI Skill，帮助你在工作中系统应用书中的思维框架。

## 一键安装

在 Cowork 中粘贴以下链接即可安装：

```
https://raw.githubusercontent.com/sunyuzheng/zhenbenshi-advisor/main/zhenbenshi-advisor.skill
```

## 这个 Skill 能做什么？

- **分析工作问题**：用书中的七大框架诊断你遇到的职场挑战
- **制定行动方案**：把"想清楚"转化为具体可执行的下一步
- **拆解复杂目标**：将大目标分解为可落地的任务序列
- **复盘与优化**：帮你反思决策过程，持续提升工作方法

## 核心框架

书中包含七大核心工作框架，Skill 会根据你的问题自动匹配最合适的框架：

1. 目标拆解框架
2. 利益相关方分析
3. 资源盘点清单
4. 风险预判矩阵
5. 执行优先级排序
6. 反馈复盘循环
7. 个人品牌建立路径

## 使用方法

安装后，直接用自然语言描述你的工作问题，Skill 会引导你完成分析和行动规划。
